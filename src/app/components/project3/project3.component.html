Bestellsystem

Ein modernes Bestellsystem, das ich als Fullstack-Projekt entwickelt habe, um das Zusammenspiel von Frontend, Backend und Datenbanken in einer containerisierten Umgebung zu verstehen. Das Projekt nutzt Angular für das UI, Node.js für die API-Logik, und kombiniert MySQL und MongoDB für unterschiedliche Datentypen. Mithilfe von Docker wurde das gesamte System modular und portabel aufgebaut – ein wichtiger Schritt in meiner praktischen Fullstack-Entwicklung.
